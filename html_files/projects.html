BritCrime Analytics

A Data End to end pipeline and analytics project that tells about the crimes in Britain.

Key Features

  • Managed to pull the data using API within 3 seconds.
  • Handled large datasets of having close to 1.9 million records.
  • Cleaned and preprocessed the data using Python libraries like Pandas and NumPy.
  • Stored the processed data in Azure Blob Storage for scalable and secure access
  • Performed exploratory data analysis (EDA) to identify crime patterns, trends, and correlations using visualization libraries like Matplotlib and Seaborn.
  • Created interactive dashboards and visualizations using Power BI to present insights to stakeholders.

Youtube Sentimental Analysis

An end to end project analyzing sentiments from YouTube comments using natural language processing techniques.

Key Features

  • Extracted YouTube comments using YouTube Data API.
